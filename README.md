# carrot

**당근마켓 클론코딩**

## 1. 개발 환경
 - Java17
 - Spring boot 3.3.4
 - Redis
 - Mysql

### 개발 세부 환경 (Library)
 - lombok
 - Spring batch
 - spring-boot-devtools
 - Mysql driver
 - JPA

 - 회원가입
 - 로그인 / 로그아웃
 - 채팅
 - 상품 관심 등록

 ### 상품
 - 상품등록
 - 상품 리스트조회
 - 상품 상세조회
 - 검색 ( 물품 및 동네 )
 - 인기 상품 리스트
